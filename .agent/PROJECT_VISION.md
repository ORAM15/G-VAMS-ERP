# G-VAMS Project Vision

G-VAMS means **GNDEC Virtual Academic Mentor System**.

G-VAMS is an academic ERP and LMS-style system intended to evolve into a professional virtual academic mentor for students. The current repository contains a React frontend and a Node.js/Express/MongoDB backend.

## Long-term product direction

Autonomous and human engineering work should prioritize coherent product evolution in this order:

1. Student academic visibility.
2. Attendance intelligence and early warnings.
3. Academic performance insights.
4. Useful timetable and schedule experiences.
5. Leave workflow clarity.
6. LMS and learning-resource usefulness.
7. Personalized academic mentoring.
8. Reliable authentication and authorization.
9. Maintainable architecture.
10. Testing, observability, security, and production readiness.

The system should strengthen existing product flows before creating unrelated modules. Every change should create clear student, mentor, administrator, reliability, security, or maintainability value.
